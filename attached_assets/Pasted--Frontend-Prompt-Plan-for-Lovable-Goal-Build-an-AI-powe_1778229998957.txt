# Frontend Prompt Plan for Lovable

Goal:

Build an AI-powered collaborative study workspace frontend only.

Target:

Modern SaaS-quality UI similar to:

* Notion

* Discord

* Quizlet

* Linear

Tech Requirements:

* Next.js 15

* TypeScript

* TailwindCSS

* shadcn/ui

* Responsive

* Dark mode default

* Clean folder structure

* Production-style UI

---

# Important Strategy

Do NOT ask Lovable to generate the entire app at once.

Best approach:

1. Generate layout system first

2. Then dashboard

3. Then notes editor

4. Then study rooms

5. Then AI features

6. Then polish

This avoids:

* messy code

* inconsistent UI

* broken architecture


# Important Architecture Instructions

Always tell Lovable:

```txt

Use:

- app router

- reusable components

- modular architecture

- server/client component separation

- TypeScript interfaces

- scalable folder structure

- clean naming conventions

```

# Recommended Frontend Libraries

Tell Lovable to use these:

| Purpose       | Library         |

| ------------- | --------------- |

| UI            | shadcn/ui       |

| Animation     | Framer Motion   |

| Charts        | Recharts        |

| Forms         | React Hook Form |

| Validation    | Zod             |

| Editor        | Tiptap          |

| Icons         | Lucide React    |

| State         | Zustand         |

| Data fetching | TanStack Query  |
